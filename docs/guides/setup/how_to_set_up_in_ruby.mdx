- As always, add a new entry to your [Gemfile](https://bundler.io/guides/gemfile.html) as the first step.

  <CodeBlock language="ruby">
    {`gem "convenient_service", "~> ${version}"`}
  </CodeBlock>


- Then run [bundle](https://bundler.io/man/bundle-install.1.html) to install the gem.

  ```bash
  bundle install
  ```

- Once finished, [require](https://ruby-doc.org/core-2.7.0/Kernel.html#method-i-require) the gem.

  ```ruby
  require "convenient_service"
  ```

- To ensure [the lib](/) is properly loaded you can define the following service.

  ```ruby
  class Service
    include ConvenientService::Standard::Config

    def result
      success
    end
  end
  ```

- And run it by the command below.

  ```ruby
  Service.result
  ```

- That's it. Happy coding!
