title: Can adjust foreign results
---

This plugin provides `result_of` instance method to your services.

It is useful to apply middlewares to results of [foreign services](/glossary#foreign_services).

For example, `result_of` is employed by [Caches Repeated Results](/plugins/service/caches_repeated_results).
